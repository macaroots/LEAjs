Clever Editing - Edição Assistida por Agentes

Agentes devem ser treinados em tempo de execução e precisam poder falhar sem a necessidade de reiniciar o sistema, literalmente matando o processo e o agente.

Experimente a Clever Editing - Edição Assistida por Agentes, direcionando eventos do DOM para um agente, por exemplo:

<button onclick="getAgent(this).see('novaAcao', 'args');">Nova ação</button>

Programação Top-Down Interativa

A edição assistida por agentes permite descrever o algoritmo numa abordagem top-down interativa. Para cada ação desconhecida o agente abre uma pergunta.

<script>
    window.agent.see('set', ['topDown', new (function () {
        this.act = async function (args, resolve, reject) {
            let agent = this.agent;
            let r1 = await agent.see('passo1', args);
            let r2 = await agent.see('passo2', r1);
            let r3;
            if (await agent.see('teste', r2)) {
                r3 = await agent.see('alternativa1', r2);
            }
            else {
                r3 = await agent.see('alternativa2', r2);
            }
            alert(r3);
            resolve(r3);
        };
    })()]);
</script>
<button onclick="getAgent(this).see('topDown', 'mundo');">CleverEditing</button>
